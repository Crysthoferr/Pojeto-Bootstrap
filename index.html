HTML

HTML (HyperText Markup Language) é a linguagem usada para estruturar e organizar o conteúdo de páginas na web.

HTML é a base da construção de páginas na internet. Ele utiliza "tags" (etiquetas) para marcar diferentes tipos de conteúdo, como textos, imagens, links e vídeos. Essas tags dizem ao navegador como exibir cada elemento na tela. HTML não é uma linguagem de programação, mas sim de marcação, usada em conjunto com CSS (para estilos) e JavaScript (para interatividade).

CSS

CSS (Cascading Style Sheets) é a linguagem usada para definir o estilo e o layout de páginas web.

CSS é responsável pela aparência visual das páginas na web. Ele permite aplicar cores, fontes, espaçamentos, bordas, tamanhos e posicionamentos aos elementos HTML. A sigla significa "Folhas de Estilo em Cascata", pois os estilos são aplicados em uma hierarquia que pode ser sobrescrita por regras mais específicas.
